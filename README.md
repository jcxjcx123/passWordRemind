# 密码备忘录

一个基于uniapp开发的密码管理应用，支持多端运行。采用分组管理模式，让您的密码管理更加有序高效。

## 功能特性

### 🔐 安全登录
- 首次使用需要设置登录密码（至少6位）
- 每次进入应用需要验证密码
- 支持修改登录密码
- 密码错误提示和安全验证
- 支持设置安全问题，用于找回密码

### 📝 密码管理
- **分组管理**：按软件名称自动分组，同一软件的多个账号统一管理
- **新增密码记录**：软件名称、用户名、密码必填，网址、邮箱、备注选填
- **编辑已有密码记录**：支持修改所有字段信息
- **删除密码记录**：支持单个账号删除，确认提示防误删
- **搜索功能**：支持按软件名称、用户名、备注、邮箱搜索

### 📋 便捷操作
- **详情查看**：点击账号项查看完整详情
- **一键复制**：支持复制用户名、密码、网址、邮箱到剪贴板
- **密码显示/隐藏**：保护隐私的同时方便查看
- **右滑操作**：快速编辑/删除账号
- **分组展开/收起**：点击分组标题控制显示状态

### 🔧 高级功能
- **分组置顶**：重要分组可置顶显示（最多5个）
- **拖拽排序**：长按分组进入交换模式，自定义排序
- **预设手机号**：设置常用手机号，新增账号时自动填充
- **数据备份**：支持导出/导入功能，数据安全有保障
- **可拖拽悬浮按钮**：新增按钮可自由拖拽到舒适位置

### 💾 本地存储
- 所有数据存储在本地，无需网络连接
- 使用uni.storage进行数据持久化
- 支持数据导出备份和恢复

## 详细操作手册

### 🚀 首次使用
1. **安装启动**：首次打开应用
2. **设置登录密码**：
   - 输入至少6位的登录密码
   - 确认密码（两次输入必须一致）
   - 点击"设置密码"完成初始化
3. **进入主界面**：设置成功后自动跳转到主界面

### 🔑 登录验证
1. **日常登录**：每次打开应用需要输入登录密码
2. **密码验证**：输入正确密码后进入主界面
3. **错误处理**：密码错误会显示提示信息

### 📝 密码管理操作

#### 新增密码记录
1. **方式一：悬浮按钮新增**
   - 点击右下角蓝色悬浮按钮（+号）
   - 填写必填信息：软件名称、用户名、密码
   - 选填信息：网址、邮箱、备注
   - 点击"保存"完成添加

2. **方式二：分组内新增**
   - 展开已有分组
   - 点击分组内的"+ 添加账号"按钮
   - 软件名称自动填充为当前分组名称
   - 填写其他信息后保存

#### 查看密码详情
1. **点击账号项**：点击任意账号条目
2. **查看完整信息**：在弹出的详情窗口中查看所有信息
3. **一键复制**：点击各字段右侧的复制按钮
4. **密码显示**：点击眼睛图标显示/隐藏密码

#### 编辑密码记录
1. **右滑操作**：在账号条目上向左滑动
2. **选择编辑**：点击蓝色"编辑"按钮
3. **修改信息**：在弹出窗口中修改需要的字段
4. **保存更改**：点击"保存"完成修改

#### 删除密码记录
1. **右滑操作**：在账号条目上向左滑动
2. **选择删除**：点击红色"删除"按钮
3. **确认删除**：在确认对话框中点击"确定"

### 🔍 搜索功能
1. **打开搜索**：点击顶部搜索框
2. **输入关键词**：支持搜索软件名称、用户名、备注、邮箱
3. **实时过滤**：输入时自动过滤显示匹配结果
4. **清空搜索**：删除搜索内容恢复完整列表

### 📁 分组管理

#### 展开/收起分组
1. **点击分组标题**：点击分组名称区域
2. **切换状态**：展开显示账号列表，收起隐藏账号
3. **记忆状态**：应用会记住每个分组的展开状态

#### 分组置顶
1. **长按分组**：长按分组标题区域
2. **选择置顶**：在弹出菜单中选择"置顶"/"取消置顶"
3. **置顶限制**：最多可置顶5个分组
4. **置顶排序**：置顶分组显示在列表顶部

#### 分组排序
1. **进入排序模式**：长按普通分组（非置顶）
2. **选择交换对象**：点击要交换到的位置
3. **完成交换**：系统自动完成位置交换
4. **退出模式**：点击右上角"退出交换"按钮
5. **注意事项**：置顶分组不参与排序交换

### ⚙️ 设置功能

#### 修改登录密码
1. **打开设置**：点击右上角设置按钮
2. **选择修改密码**：点击"修改密码"选项
3. **输入原密码**：验证当前登录密码
4. **设置新密码**：输入新密码并确认
5. **设置安全问题**（可选）：
   - 输入安全问题（用于找回密码）
   - 输入安全问题答案
   - 如不需要可留空
6. **完成修改**：保存后下次登录使用新密码

#### 预设手机号
1. **打开设置**：点击右上角设置按钮
2. **选择预设手机号**：点击"预设手机号"选项
3. **输入手机号**：输入常用的手机号码
4. **保存设置**：新增账号时自动填充此手机号

### 💾 数据备份

#### 导出数据
1. **打开备份**：点击顶部备份按钮
2. **选择导出**：点击"导出数据"按钮
3. **保存文件**：选择保存位置，文件名自动生成
4. **完成导出**：JSON格式文件包含所有密码数据

#### 导入数据
1. **打开备份**：点击顶部备份按钮
2. **选择导入**：点击"导入数据"按钮
3. **选择文件**：选择之前导出的JSON文件
4. **确认导入**：数据将合并到现有记录中

### 🎯 界面操作技巧

#### 悬浮按钮
1. **拖拽移动**：长按悬浮按钮可拖拽到任意位置
2. **点击新增**：轻点悬浮按钮快速新增密码
3. **位置记忆**：应用会记住按钮的位置

#### 复制功能
1. **详情页复制**：在详情页点击字段右侧复制按钮
2. **复制反馈**：复制成功会显示提示信息
3. **支持字段**：用户名、密码、网址、邮箱均可复制

#### 震动反馈
1. **长按反馈**：长按操作时提供震动反馈
2. **操作确认**：重要操作完成时的震动提示
3. **系统设置**：可在系统设置中关闭震动

## 常见问题

### ❓ 使用问题

**Q: 忘记登录密码怎么办？**
A: 如果您设置了安全问题，可以通过以下步骤找回密码：
- 在登录页面点击"忘记密码"按钮
- 正确回答您设置的安全问题
- 设置新的登录密码

如果未设置安全问题，建议：
- 卸载重装应用（会丢失所有数据）
- 如有数据备份文件，重装后可导入恢复

**Q: 为什么搜索不到某些记录？**
A: 搜索功能支持以下字段：
- 软件名称、用户名、备注、邮箱
- 不支持搜索密码内容（安全考虑）

**Q: 分组排序失效怎么办？**
A: 分组排序注意事项：
- 置顶分组不参与排序交换
- 只能在普通分组之间进行排序
- 长按普通分组进入交换模式

**Q: 数据导入失败怎么办？**
A: 检查导入文件：
- 确保是本应用导出的JSON文件
- 文件格式正确且未损坏
- 重新尝试导出和导入操作

### ⚠️ 注意事项

1. **数据安全**
   - 定期备份重要数据
   - 妥善保管备份文件
   - 避免在不安全的设备上使用

2. **密码设置**
   - 登录密码建议设置复杂一些
   - 不要使用过于简单的密码
   - 定期更换登录密码

3. **使用建议**
   - 重要账号建议额外备份
   - 不要在公共场所查看密码
   - 及时关闭应用避免他人查看

4. **系统兼容**
   - 建议使用最新版本的操作系统
   - 确保设备有足够的存储空间
   - 定期清理应用缓存

   ```bash
   3. 自动打开浏览器预览
   ```

2. **微信小程序**
